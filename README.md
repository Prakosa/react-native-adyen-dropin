# react-native-adyen-dropin

React Native bridge for Adyen drop-in

## Installation

```sh
npm install react-native-adyen-dropin
```

## Usage

```js
import AdyenDropin from "react-native-adyen-dropin";

// ...

const result = await AdyenDropin.multiply(3, 7);
```
